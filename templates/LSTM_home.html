Energy Forecasting API

This is a simple API that returns forecasts of capacity and production of energy in Colombia

The API was build with flask python library and the models implement until now were trained with tensorflow and prohet

JSON

                            
                                {
                                    "success":True,
                                    "prediction": "The prediction is -n_ahead- days in the future"
                                    "response": [456.7, 985.9]
                                }
